# HW2 – Machine Learning Model Comparison

## Overview
This project focuses on comparing different machine learning models for classification tasks and analyzing their performance.

## Models Implemented
- Neural Networks
- Random Forest
- Support Vector Machine (SVM)

## Approach
- Train multiple models on the same datasets
- Compare performance across models
- Analyze model behavior and generalization

## Features
- Model evaluation using accuracy and validation techniques
- Hyperparameter tuning for performance optimization
- Analysis of overfitting and bias-variance trade-offs
- Feature engineering to improve model performance

## Key Learnings
- Understanding strengths and weaknesses of different ML models
- Impact of model complexity on performance
- Importance of proper feature design and tuning
- Trade-offs between interpretability and performance

## Results
- Identification of best-performing models for different datasets
- Demonstrated impact of hyperparameter tuning
- High classification accuracy across multiple models

## Tech Stack
- Python
- scikit-learn
- NumPy
- Matplotlib
